# Samurai Backend
Using [Node.js](https://nodejs.org/) and [Express](https://expressjs.com).
## Project Setup
```
npm install
```
## Database Connection Configuration
Create a file named ```.env```.
File should contain:
```
USER = [user]
PASSWORD = [password]
CONNECTIONSTRING = "COT-CIS4375-03.cougarnet.uh.edu/orclpdb"
```
### Project Start
```
npm start
```
